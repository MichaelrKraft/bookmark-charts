How do I export my bookmarks?
  1. Chrome / Brave / Edge: chrome://bookmarks → ⋮ menu → Export bookmarks → save as bookmarks.html
  2. Firefox: Ctrl+Shift+O → Import and Backup → Export Bookmarks to HTML
  3. Safari: File → Export → Bookmarks (saves as Bookmarks.html)
  4. Arc: Arc menu → Archive → Export Bookmarks
